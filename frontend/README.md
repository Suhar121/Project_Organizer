# Frontend

Vue 3 + TypeScript + Vite frontend for Dev Dashboard.

## Scripts

- `npm run dev` - start local dev server
- `npm run build` - type-check + production build
- `npm run preview` - preview production build locally

## Local Development

```bash
npm install
npm run dev
```

The frontend expects the backend API at `http://localhost:6001`.

## Notes

- UI components live in `src/components/`
- Main dashboard page is `src/pages/Dashboard.vue`
- API client is `src/services/api.ts`
